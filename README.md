# Duplicate Question Detection on Quora
My first Machine Learning project which implements Random Forest Classifier to classify the questions as duplicate or not. 
The dataset downloaded from kaggle: "https://www.kaggle.com/datasets/thedevastator/quora-duplicate-questions-detection". 
The dataset consist of 6 attributes - id, qid1, qid2, question1, question2, and is_duplicate.
TfiidfVectorizer has been implemented to convert string into numerical format. 
The model gives 57% accuracy.
Further implementation of other vectorization technique could be implemented to get higher accuracy.
